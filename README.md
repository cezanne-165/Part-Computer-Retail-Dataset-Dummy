##  Metodologi
Analisis dilakukan menggunakan:
- Python
- Jupyter Notebook (Anaconda)
- Algoritma Apriori (`apyori`)

Tahapan analisis meliputi:
1. Preprocessing data transaksi
2. Pembentukan transaksi (basket analysis)
3. Pencarian frequent itemsets
4. Pembentukan association rules
5. Evaluasi aturan menggunakan:
   - Support
   - Confidence
   - Lift
6. Visualisasi hasil analisis
Tidak ada data pribadi atau data sensitif dalam dataset ini.
